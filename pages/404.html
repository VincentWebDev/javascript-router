404 Not Found

The requested page or route doesn't exist.

If you're rendering this JavaScript router code locally or with VSCode live server, and you refresh the brower on a URL route that is not the index page, you will receive a 404 error (Cannot GET).

I'm hosting the app on Netlify's free account. I created a netlify.toml file to handle 404 errors.

Below is the Netlify code to redirect all 404 url router errors to index.html. This code overwrites Netlify's own 404 reponse. Normally, you would need to write similar code in a .htaccess file.

[[redirects]]
from = "/*"
to = "/index.html"
status = 404
